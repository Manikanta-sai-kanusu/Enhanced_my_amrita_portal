# 📖 About The Project

MyAmrita++ is a comprehensive University ERP platform developed to streamline and digitize the complete academic and administrative ecosystem of an educational institution.

The application provides an integrated platform where students, faculty members, administrative officers, hostel wardens, and chief wardens can efficiently manage day-to-day academic and institutional activities from a single secure portal.

The system eliminates manual paperwork by automating attendance management, academic records, hostel administration, fee processing, gate pass approvals, feedback collection, assignment handling, counselling requests, document verification, and various student services.
